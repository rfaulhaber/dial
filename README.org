*** [ ] Ratio type
- Syntax
  #+begin_src
  22/7
  #+end_src
- Automatically reduces
- =ratio?= function
- =numerator= function
  #+begin_src
  (numerator 22/7) ; => 22
  #+end_src
- =denominator= function
  #+begin_src
  (denominator 22/7) ; => 7
  #+end_src
